# 画像生成プロンプト集

---

## 1. 画像生成

### プロンプト

```
近未来の東京・渋谷スクランブル交差点の夜景を生成してください。

【シーン設定】
- 場所：渋谷スクランブル交差点
- 時間帯：深夜
- スタイル：サイバーパンク
- 雰囲気：ネオンが反射する濡れた路面、人々の往来、巨大な日本語看板や広告ビジョン

【技術仕様】
- アスペクト比：16:9
- 画質：超高解像度・映画的クオリティ
- 光の表現：ネオンサイン・ホログラム広告・街灯・車のヘッドライトによる複雑な光と影

【重要な要素】
- 日本語テキストを含む看板・広告（実在しない架空のブランド名でよい）
- 近未来的な乗り物や街の改変要素を一部含める
- 写実的でありながらもSFのビジュアルスタイル
```

### 評価ポイント

- **日本語テキストの再現性**：看板・広告の日本語文字が正確に描画されているか（文字化けや崩れがないか）
- **光の表現**：ネオンのグロー・路面への反射・複数光源の重なりが自然に描かれているか
- **映画的クオリティ**：構図・被写界深度・全体的な完成度が映画のワンシーンレベルか
- **サイバーパンクの再現度**：指定スタイルの雰囲気が適切に表現されているか
