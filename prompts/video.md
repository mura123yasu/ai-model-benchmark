# 動画生成プロンプト集

---

## 1. 動画生成

### プロンプト

```
コーヒーカップにコーヒーが注がれる瞬間の動画を生成してください。

【シーン設定】
- 被写体：白いセラミックのコーヒーカップ
- 動作：上からコーヒーがゆっくりと注がれる
- 演出：スローモーション（実際の速度の1/4程度）

【技術仕様】
- 長さ：3〜5秒
- カメラアングル：やや斜め上から（カップ全体と注がれる様子が見える角度）
- 映像品質：広告・CMレベルの高品質映像
- 背景：ダークトーンの木製テーブル、柔らかいボケ

【重要な要素】
- 液体の流れとカップ内への衝突の物理的整合性
- コーヒーの色・透明度・泡立ちの自然な表現
- 蒸気のわずかな描写
- ハイライトとシャドウのリッチな光の表現
```

### 評価ポイント

- **液体の物理的整合性**：コーヒーが注がれる動きが重力・粘性を自然に再現しているか（不自然なループや歪みがないか）
- **光の自然さ**：液体の透明感・光沢・反射が広告品質で描かれているか
- **滑らかさ**：スローモーション演出でフレームが滑らかに補間されているか
- **広告品質**：全体的な映像クオリティが商業広告として使用できるレベルか
